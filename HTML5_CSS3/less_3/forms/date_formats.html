На этой странице основными нововведениями являются временные типы для атрибута type.

type="date"
Используется для ввода пользователем даты. Включает в себя год, месяц и день. Отправляет на сервер строку в формате: 2017-02-25.
type="time"
Используется для ввода пользователем времени в формате чч:мм. Отправляет на сервер строку в таком же формате.
type="datetime-local"
Используется для ввода пользователем времени в формате дд.мм.гггг чч:мм. Часовой пояс не учитывается. Отправляет на сервер строку в формате гггг-мм-ддTчч:мм, например 2017-02-18T10:15.
type="month"
Позволяет выбрать месяц и год. На сервер данные приходят в формате гггг-мм, например: 2017-02.
type="week"
Позволяет выбрать год и неделю. На сервер данные отправляются в формате гггг-Wнн. Например: 2017-W08.
