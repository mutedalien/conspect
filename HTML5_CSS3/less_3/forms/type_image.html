На этой странице представлен type="image".

type="image"
Представляет собой картинку, кликая по которой пользователь отправляет координаты места клика на картинке. Данные приходят на сервер в формате двух параметров x и y. Например: x:60 y:8. Картинка является одновременно кнопкой, которая отправляет форму.
