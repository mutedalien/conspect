На этой странице представлены другие популярные типы для атрибута type.

type="email"
Используется для ввода пользователем корректного email-адреса. Браузер требует чтобы пользователь ввел данные в формате адрес_ящика@сервер.
type="tel"
Используется для ввода пользователем телефонного номера. Специальных требований по оформлению нет.
type="url"
Используется для ввода пользователем корректного url. Браузер требует чтобы ползователь ввел данные в формате http://domain.zone.
type="number"
Используется для ввода пользователем корректного числа. Браузер ограничивает ввод только числовыми значениями.
type="color"
Используется для отправки кода цвета. Отправляет значение цвета в шестнадцатеричном формате, например #80ff00.
