Ознакомительный фрагмент аудио.

Допустимые форматы файлов: mp3, ogg, wav.

Тег audio может содержать следующие атрибуты:
  1. autoplay - аудиофайл начнет проигрываться сразу после загрузки страницы
  2. controls - атрибут "включает" элементы управления
  3. loop - атрибут заставляет файл проигрываться снова и снова
  4. muted - выключает звук
  5. src - указывает на расположение аудиофайла
  6. preload - управляет предварительной загрузкой файла и может иметь следующие значения:
    • none - аудиофайл не будет предварительно загружен на страницу
    • metadata - только данные о файле будут загружены, например его длительность
    • auto - файл будет целиком загружен вместе со страницей, даже если пользователь не собирался его использовать заранее
    Отсутствие значения для атрибута является синонимом значения auto.

Также аудио запись может быть загружена с использованием тега source. При загрузке страницы браузер самостоятельно выберет подходящий файл.
