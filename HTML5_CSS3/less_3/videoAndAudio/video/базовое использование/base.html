Базовое использование тега video

Поддерживаемые форматы: mp4, webm, ogg.

Тег может содержать следующие атрибуты:

  1. autoplay - видео начнет воспроизводиться после загрузки страницы
  2. controls - добавляет элементы управления видеозаписью
  3. height - значение в пикселях, устанавливает высоту плеера
  4. loop - устанавливает повторение воспроизведения файла снова и снова
  5. muted - отключает звук
  6. poster - значение: url, устанавливает изображение в случае если видео долго загружается или если пользователь не нажал кнопку воспроизведения
  7. preload - управляет предварительной загрузкой файла и может иметь следующие значения:
    • none - видео не будет предварительно загружено на страницу
    • metadata - определяет что только метаинформация должна быть загружена, например длительность видеозаписи
    • auto - определяет что видеозапись должна быть целиком загружена, даже если пользователь не собирался ее смотреть
    Пустое значение - синоним значения auto.
