Пример с использованием субтитров.

Тег track - позволяет прикреплять файл с субтитрами к видеозаписи.

Может иметь следующие атрибуты:
  1. src - расположение файла с субтитрами
  2. default - устанавливает какой файл субтитров должен использоваться по умолчанию
  3. kind - устанавливает назначение файла с субтитрами и может иметь следующие значения
    • subtitles
      - предоставляют перевод содержимого видеофайла, которое может быть не понято зрителем, например фильм на русском языке, но там есть часть где персонажи говорят на английском, как раз для такого случая будет уместно использовать этот тип.
      - также этот тип может быть использован для указания дополнительной информации, например для указания даты когда происходят события на экране или для указания местополжения текущих событий.
    • captions
      - могут предоставлять транскрипцию, а также перевод речи на язык зрителя
      - могут содержать важную не вербальную информацию например о том какая сейчас играет музыка или какие звуковые эффекты происходят. Также могут содержать информацию об авторских правах песни, тексте песни, исполнителе.
      - подходит для глухих зрителей, а также в случаях когда звук выключен.
    • descriptions - текстовое описание текущего видеоряда
    • chapters - используется для обозначения заголовков разделов, когда пользователь осуществляет навигацию по видеофайлу.
    • metadata - информация для скриптов, не видна пользователю
  4. label - определяет назначения для файла с субтитрами
  5. srclang - язык на котором написаны субтитры, перечень валидных кодов можно посмотреть здесь: https://r12a.github.io/apps/subtags/. Если атрибут kind="subtitles", тогда атрибут srclang должен быть обязательно иметь значение.

Субтитры пишутся в следующем формате: 

        WEBVTT

        00:00:10.000 --> 00:00:12.500
        надпись выводится в период с 10 сек. по 12 сек

        00:00:13.200 --> 00:00:16.900
        надпись выводится в период с 13 по 16 секунду

        00:00:17.200 --> 00:01:05.800
        <p>Данный текст показывает возможность использования</p>
        <p>многострочных субтитров </p>
        <p>Созданных при помощи HTML тегов</p>

Валидатор субтитров: https://quuz.org/webvtt/
